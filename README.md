# SHERLOC Spacesui Material Study
Python Jupyter Notebooks for processing and analysis of DUV Raman spectra of spacesuit materials on Mars

These Notebooks process Raman & fluorescence spectra and spectral maps acquired using the MOBIUS and SHERLOC instruments.

 - SRLC_CalTargets.ipynb: Notebook for processing SHERLOC spectra, comparing them to reference spectra, and doing chemical kinetics fitting

Any updates to these Notebooks will be made available online at www.github.com/Jobium/SHERLOC_spacesuit_materials/

Last updated by Dr Joseph Razzell Hollis on 2026-03-03. Tested on Python 3.13.5 and JupyterLab 4.3.4.

These scripts require Python 3.13 and the following packages: os, math, glob, datetime, numpy, pandas, matplotlib, lmfit, itertools, scipy. The SRLC_RF-mapping script requires pre-processed data files generated by publicly available Loupe software, which handles raw SHERLOC data.
